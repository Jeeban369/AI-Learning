### **1. Eigenvalues рмПрммрмВ Eigenvectors**  
**Eigenvalue (╬╗)** рмПрммрмВ **Eigenvector (v)** рм╣рнЗрмЙрмЫрмирнНрмдрм┐ рморнНрнЯрм╛рмЯрнНрм░рм┐рмХрнНрм╕рнН рм╕рм╣рм┐рмд рмЬрмбрм┐рмд рмПрмХ рмЧрм╛рмгрм┐рмдрм┐рмХ рм╕рморнНрмкрм░рнНрмХ:

- **A:** рморнНрнЯрм╛рмЯрнНрм░рм┐рмХрнНрм╕рнН

- **╬╗:** Eigenvalue (рм╕рнНрмХрнЗрм▓рм╛рм░рнН)

### **2. Singular Value Decomposition (SVD)**  
SVD рм╣рнЗрмЙрмЫрм┐ рмпрнЗ рмХрнМрмгрм╕рм┐ рморнНрнЯрм╛рмЯрнНрм░рм┐рмХрнНрм╕рнН M рмХрнБ рмдрм┐рмирнЛрмЯрм┐ рморнНрнЯрм╛рмЯрнНрм░рм┐рмХрнНрм╕рнН рм░рнЗ рммрм┐рмнрмХрнНрмд рмХрм░рм╛рмпрм╛рмП:  
\[ M = U \cdot \Sigma \cdot V^T \]  
- **U рмПрммрмВ V:** Orthogonal matrices (рмШрнВрм░рнНрмгрнНрмгрми рмПрммрмВ рмкрнНрм░рмдрм┐рмлрм│рми рм░рм┐рмкрнНрм░рнЗрмЬрнЗрмгрнНрмЯрнН рмХрм░рмирнНрмдрм┐)ред  
- **╬г:** Diagonal matrix (singular values, ╧ГтВБ тЙе ╧ГтВВ тЙе ... тЙе 0)ред  

**рмЙрмжрм╛рм╣рм░рмг 2:** SVD рмХрм╛рмврм┐рммрм╛  
```python  
M = np.array([[1, 2], [3, 4], [5, 6]])  
U, ╬г, VT = np.linalg.svd(M)  

print("U:\n", U)  
print("Singular Values:", ╬г)  # Output: [9.52551809 0.51430058]  
print("VT:\n", VT)  
```
